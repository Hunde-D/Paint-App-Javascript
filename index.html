Quick Guid

  • fill btn on click fills the shape on dbclick strokes the shape
  • stroke btn onclick stroke the shape
  • Group btn groups selected objects together like one object
  • selectable btn makes all object on the canvas selectable
  • unselectable btn makes the active object(selected) unselectable
  • multiselect btn selects all the objects on the canvas,but it won't group them
  • discard btn unselects the selected(active) object
  • remove btn removes the active object from the canvas
